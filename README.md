# P2P Lending Platform

## Project Description
A decentralized peer-to-peer (P2P) lending platform built with Solidity, allowing borrowers to request loans and lenders to fund them securely on the blockchain.

## Project Vision
To create a transparent, trustless, and secure system where individuals can lend and borrow funds without intermediaries.

## Key Features
- **Loan Requesting**: Borrowers can create loan requests with desired amounts.
- **Loan Funding**: Lenders fund loans directly using Ethereum.
- **Repayment System**: Borrowers can repay loans, automatically updating loan status.
- **Smart Contract Transparency**: All transactions and requests are handled on-chain.

## Future Scope
- Implement interest rate and repayment schedule.
- Add collateral mechanism.
- Allow lender-borrower rating system.
- Integrate frontend dashboard for interactions.
- Add stablecoin support for less volatility.
